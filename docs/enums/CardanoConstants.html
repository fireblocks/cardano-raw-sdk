    MAX_TX_INPUTS: 100

    Maximum number of UTxO inputs per transaction. Cardano's 16KB transaction size limit allows ~100–150 inputs in practice. This cap prevents dust-attack vectors where many tiny UTxOs are sent to an address to make future transactions exceed the size limit.
